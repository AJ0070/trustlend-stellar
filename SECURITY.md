# Security Policy

## Supported Versions

Currently, the following versions of the project are being supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.1.x   | :white_check_mark: |

## Reporting a Vulnerability

We take the security of TrustLend very seriously. 

If you discover a security vulnerability within TrustLend, please do NOT open a public issue. Instead, please send an e-mail to the maintainers privately.

All security vulnerabilities will be promptly addressed. We will respond to your report as quickly as possible, usually within 48 hours. If the vulnerability is confirmed, we will work with you to patch it and release an update before disclosing it publicly.

Thank you for helping keep TrustLend secure!
